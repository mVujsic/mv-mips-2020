### Покретање програма
1. **сместити .аsm фајл репоа у фолдер са изнад наведеним (2) (3).**
2. **покренути DosBox** 
3. **секвенцијално покренути следеће наредбе**
``` bat
mount c: <путања_до_фолдера>
c:
masm /a <име>.ASM
link <име>.OBJ
<ime>.EXE
```
